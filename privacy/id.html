Kebijakan Privasi

Pembaruan terakhir: 29 Juni 2025

Kami berkomitmen untuk melindungi privasi Anda. Game seluler ini dirancang untuk mengumpulkan hanya informasi yang diperlukan untuk fitur permainan dasar, menampilkan iklan berhadiah dengan persetujuan Anda, dan mempertahankan papan peringkat global.

ğŸ” Apa yang Kami Kumpulkan

1. Autentikasi Anonim

  • Kami menggunakan Firebase Anonymous Authentication untuk membuat ID pengguna unik dan acak per perangkat.
  • ID ini tidak terikat pada nama asli, email, atau informasi pribadi lainnya.
  • ID ini hanya digunakan untuk menyimpan progres dan mengirim skor ke papan peringkat.

2. Partisipasi Papan Peringkat

  • Skor tertinggi Anda disimpan di Google Firestore, hanya terhubung ke ID anonim dan nama acak yang dibuat dalam aplikasi.
  • Kami tidak menyimpan informasi pribadi lainnya bersama entri papan peringkat.

3. Iklan Berhadiah (AdMob)

  • Iklan video berhadiah disediakan melalui Google AdMob.
  • Iklan hanya ditampilkan setelah Anda memberikan persetujuan eksplisit melalui ATT (Apple) dan UMP (Google).
  • Jika Anda mengizinkan iklan yang dipersonalisasi, AdMob dapat mengakses Identifier Iklan (IDFA) perangkat Anda.
  • Kami tidak mengumpulkan, mengakses, atau menyimpan ID iklan atau pelacakan apa pun sendiri.

4. Data Perangkat Lokal

  • Beberapa pengaturan dan data sementara disimpan secara lokal di perangkat Anda.
  • Ini termasuk progres permainan, preferensi UI, dan pilihan privasi Anda.

5. Data Penggunaan

  • Kami tidak melacak perilaku permainan, durasi sesi, atau penggunaan perangkat Anda.
  • AdMob dapat mengumpulkan data interaksi iklan anonim (misalnya tayangan, tampilan, penyelesaian) untuk menayangkan iklan dengan benar.
