Sekretesspolicy

Senast uppdaterad: 29 juni 2025

Vi är engagerade i att skydda din integritet. Det här mobilspel är utformat för att endast samla in den information som krävs för att möjliggöra viktiga spelkomponenter, visa belönade annonser med ditt samtycke och upprätthålla en global topplista.

🔐 Vad vi samlar in

1. Anonym autentisering

  • Vi använder Firebase Anonymous Authentication för att generera ett unikt, slumpmässigt användar-ID per enhet.
  • Detta ID är inte kopplat till ditt riktiga namn, e-postadress eller andra personliga uppgifter.
  • Det används endast för att spara dina framsteg och möjliggöra topplisteinlägg.

2. Topplistedeltagande

  • Din poäng lagras i Google Firestore, endast kopplat till ditt anonyma ID och ett slumpmässigt visningsnamn som genereras i appen.
  • Vi lagrar inte någon annan personlig information tillsammans med topplisteposter.

3. Belönade annonser (AdMob)

  • Belönade videoannonser tillhandahålls via Google AdMob.
  • Annonsvisning sker endast efter att du gett ditt uttryckliga samtycke via Apples ATT och Googles UMP.
  • Om du tillåter personliga annonser kan AdMob få tillgång till din enhets annonsidentifierare (IDFA).
  • Vi samlar inte in, har inte tillgång till eller lagrar några annons- eller spårningsidentifierare själva.

4. Lokala enhetsdata

  • Vissa inställningar och tillfälliga speldata kan lagras lokalt på din enhet.
  • Detta inkluderar spelstatus, UI-inställningar och val för sekretessgodkännande.

5. Användningsdata

  • Vi spårar inte ditt spelbeteende, sessionens längd eller enhetsanvändning.
  • AdMob kan samla in anonym interaktionsdata (t.ex. visningar, fullföljanden) för att kunna visa annonser korrekt.

🧩 Tredjepartstjänster

  • Google Firebase – anonym inloggning och lagring av topplistor
  • Google AdMob – valfria belönade annonser och samtyckeshantering
  • Apple (iOS) – hanterar sekretesspromptar via ATT och spårningskontroll

🚫 Vad vi inte samlar in

  • Vi samlar inte in ditt riktiga namn, e-postadress eller telefonnummer.
  • Inget konto eller lösenord krävs eller stöds.
  • Vi spårar inte din plats.
  • Vi har inte tillgång till dina kontakter, bilder eller andra appar.
  • Vi utför inte beteendeanalys eller användarprofilering.
  • Vi använder inte cookies eller fingeravtryckstekniker.

🛠 Dina val och kontroller

  • Du kan neka eller återkalla spårningstillstånd när som helst.
  • Gå till Inställningar → Sekretess på din iOS-enhet för att hantera sekretessinställningar.
  • I samma avsnitt kan du också radera appdata eller återställa spårningsidentifierare.
  • Du kan radera appen när som helst för att permanent stoppa all datainsamling.

📅 Databevarande

  • Ditt anonyma ID och topplistepoäng behålls tills de tas bort manuellt eller automatiskt under appuppdateringar.
  • Ingen känslig data lagras någonsin.
